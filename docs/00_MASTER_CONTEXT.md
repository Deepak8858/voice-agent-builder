# 00 — Master Context

## Vision
Build a Lovable-style SaaS platform for AI voice agents. Users describe their desired agent, upload business data, connect integrations, test in a browser, deploy to phone numbers or workspaces, and monitor real outcomes.

## Main User Flow
```txt
Signup → Create workspace → Choose template or describe need → Generate Agent Spec JSON → Edit in chat or visual builder → Upload knowledge → Connect tools → Test call → Publish → Analyze calls → Improve agent
```

## Product Pillars
- Lovable-style generation
- Vertical templates
- Compliance-first calling
- Provider-agnostic runtime
- White-label agency system
- Analytics and evaluations

## First Ideal Customer
AI agencies and automation freelancers. They need white-label voice agents they can sell repeatedly to clients.

## First Ideal Use Cases
- AI receptionist
- Dental/clinic receptionist
- Real estate lead qualification
- Appointment reminders
- D2C order confirmation
- Missed-call recovery

## Key Product Truth
Do not compete as a generic voice API. Compete as the fastest way to create, test, deploy, and sell reliable business voice agents.
