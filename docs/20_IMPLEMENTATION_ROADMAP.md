# 20 — Implementation Roadmap

## Phase 0 — Setup
Monorepo, Next.js app, NestJS API, PostgreSQL, Redis, auth, CI/CD.

## Phase 1 — Agent Builder
Agent tables, Agent Spec schema, prompt-to-agent mock generator, agent builder UI, versioning.

## Phase 2 — Templates and Knowledge
MVP templates, template selector, manual FAQ, PDF/CSV upload, chunking and embeddings.

## Phase 3 — Voice Runtime
Voice adapter interface, mock provider, browser test session, Vapi/Retell adapter, call events/transcripts.

## Phase 4 — Inbound Deployment
Publish agent, provider runtime ID, inbound webhook, call log, post-call evaluation.

## Phase 5 — Integrations
Tool registry, webhook tool, Google Calendar, tool logs.

## Phase 6 — Compliance
Contacts, consent, DNC/DND, opt-out, compliance checks, audit logs.

## Phase 7 — Analytics
Event ingestion, dashboard metrics, agent metrics, improvement suggestions.

## Phase 8 — White Label
Client workspaces, branding, client dashboard, agency dashboard.

## Phase 9 — Billing
Stripe checkout, subscription webhooks, usage metering.

## Phase 10 — Production Hardening
Security review, rate limits, observability, backups, load testing.
