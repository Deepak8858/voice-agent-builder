# 01 — Product Requirements

## Problem
Businesses miss calls and follow-ups. Building reliable AI voice agents is technical, fragmented, and risky. Agencies want to sell voice agents but need a white-label platform.

## Solution
A SaaS platform where users describe a business need and the system generates a complete voice agent with persona, voice style, call flow, required fields, knowledge base, tools/integrations, compliance settings, test scenarios, and analytics goals.

## Personas
| Persona | Job |
|---|---|
| AI agency owner | Sell voice agents to clients under own brand |
| SMB owner | Capture calls, book appointments, reduce missed leads |
| Operations manager | Automate repetitive calls and track outcomes |
| Developer/implementer | Integrate tools and debug workflows |
| End caller | Get fast help without waiting |

## Core Features
1. Workspace and organization management
2. Prompt-to-agent generation
3. Vertical templates
4. Agent Spec JSON editor/validator
5. Visual call-flow builder
6. Knowledge upload and RAG
7. Voice test playground
8. Inbound deployment
9. Controlled opt-in outbound calls
10. Compliance engine
11. Integrations and tool registry
12. Call logs, transcripts, recordings
13. Analytics dashboard
14. White-label agency dashboard
15. Stripe billing and usage tracking

## Non-Goals for MVP
- Full custom SIP runtime
- Cold outbound blasting
- Enterprise SSO
- Template marketplace
- Complex call-center workforce management
- Regulated advice workflows

## Success Metrics
- First agent created in under 10 minutes
- Test call works without developer help
- 80%+ template test calls complete expected goal
- 100% outbound calls pass compliance gate
- Call analytics visible after call end
- Agency can create client workspace and branded dashboard
